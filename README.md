Anarchive is a Vite + React static site scaffold for an infinite “crazy wall” archive.

## Development

- `npm install`
- `npm run dev`

## Build

- `npm run build`
- `npm run preview`

## Hosting

This project targets GitHub Pages and DreamHost. The Vite base path is set to `/` for root deployment on GitHub Pages.
